## 👤 Who I Am

I'm **Briton Kiplangat** — a self-taught full-stack developer and SaaS builder from 🇰🇪 Kenya. I specialize in building production-grade **WhatsApp automation systems**, **SaaS platforms**, scalable **Node.js backends**, and **Linux/VPS infrastructure** that powers real communities at scale.

My open-source **WhatsApp bot framework** ([silentwolf](https://github.com/WOLFTECH-254/silentwolf)) has been forked **100+ times** and is actively deployed across developer communities throughout Africa. My latest project, **[wolfXmonitor](https://github.com/WOLFTECH-254/wolfXmonitor)**, is a full-stack uptime monitoring SaaS live at [monitor.xwolf.space](https://monitor.xwolf.space). I don't just write code — I ship tools that get adopted.

Beyond automation, I've built an entire developer ecosystem: a **Heroku-style hosting platform** with native **M-Pesa payment integration**, core database libraries, and complete documentation systems. I'm currently open to **junior developer roles**, **backend contracts**, and **freelance automation projects**.

> *"I build systems that work while you sleep."*

### 📡 [wolfXmonitor](https://github.com/WOLFTECH-254/wolfXmonitor) — Open-Source Uptime Monitoring SaaS

A full-stack **uptime monitoring SaaS platform** that pings your websites and APIs every minute and alerts you by email the instant they go down. Built with a dark green aesthetic, it includes multi-user support, **Free and Pro subscription tiers** via Paystack (M-Pesa), Brevo email alerts, a public status page, and a full admin control panel — live at [monitor.xwolf.space](https://monitor.xwolf.space).
